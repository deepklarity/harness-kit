Build a Pomodoro timer as a single-page web app. No frameworks, just HTML/CSS/JS.

25 min work, 5 min short break, 15 min long break (after every 4 work sessions). Start/pause/reset controls. The timer should show minutes:seconds and update the page title so you can see it in the browser tab.

When a session ends, play a short beep (use the Web Audio API, no sound files) and auto-switch to the next session type. Show which session you're on (e.g. "Work 2/4").

Keep it minimal but make it look good — dark background, large readable timer, color shifts between work (warm) and break (cool). Should feel calm, not cluttered.

On a second screen show a simple log of completed sessions (e.g. "Work session 1 completed at 10:30 AM"). This log should update in real-time as sessions complete.
