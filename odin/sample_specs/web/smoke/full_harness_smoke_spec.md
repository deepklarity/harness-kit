# Full Harness Smoke Test

Create a file `smoke_output/report.html` — a single HTML page containing 6 styled data tables, one generated by each agent. Each table should have 4 columns (Name, Role, Score, Notes) and 3 rows of fictional data. Use inline CSS with distinct color themes per table.

## Tasks

1. **Qwen table** — Agent: qwen. Create `smoke_output/qwen_table.html` containing a single HTML table with 4 columns (Name, Role, Score, Notes), 3 rows of fictional data, and inline CSS with a blue color theme. Include a heading identifying the agent.
2. **Gemini table** — Agent: gemini. Create `smoke_output/gemini_table.html` containing a single HTML table with 4 columns (Name, Role, Score, Notes), 3 rows of fictional data, and inline CSS with a green color theme. Include a heading identifying the agent.
3. **Claude table** — Agent: claude. Create `smoke_output/claude_table.html` containing a single HTML table with 4 columns (Name, Role, Score, Notes), 3 rows of fictional data, and inline CSS with a purple color theme. Include a heading identifying the agent.
4. **MiniMax table** — Agent: minimax. Create `smoke_output/minimax_table.html` containing a single HTML table with 4 columns (Name, Role, Score, Notes), 3 rows of fictional data, and inline CSS with an orange color theme. Include a heading identifying the agent.
5. **Codex table** — Agent: codex. Create `smoke_output/codex_table.html` containing a single HTML table with 4 columns (Name, Role, Score, Notes), 3 rows of fictional data, and inline CSS with a red color theme. Include a heading identifying the agent.
6. **GLM table** — Agent: glm. Create `smoke_output/glm_table.html` containing a single HTML table with 4 columns (Name, Role, Score, Notes), 3 rows of fictional data, and inline CSS with a teal color theme. Include a heading identifying the agent.
7. **Assemble report** — Agent: gemini. Depends on: 1, 2, 3, 4, 5, 6. Read all 6 HTML table files from `smoke_output/` and combine them into a single `smoke_output/report.html` page. Add a title "Full Harness Smoke Test Report" and arrange tables vertically with agent name headings. Rate each table's CSS quality out of 10 in a summary section at the bottom.
